# Инструкция по установке кастомной панели расширений в Firefox

Данный скрипт заменяет стандартную кнопку расширений (Unified Extensions) Firefox на функциональную панель, которая позволяет:
- Быстро включать/отключать расширения с помощью iPhone-style переключателей.
- Закреплять/откреплять расширения на панели инструментов (кнопка 📌).
- Запоминать состояние закрепления для отключенных расширений (при повторном включении расширение автоматически закрепится на панели).
- Удалять расширения прямо из панели (кнопка 🗑).
- Разделять расширения по 4 категориям:
  1. Включенные и закрепленные.
  2. Включенные и незакрепленные.
  3. Отключенные, которые были закреплены.
  4. Отключенные и незакрепленные.

---

## Шаг 1: Подготовка профиля Firefox

1. Откройте Firefox и введите в адресную строку `about:support`. Нажмите Enter.
2. В таблице найдите строку **«Папка профиля»** (Profile Folder) и нажмите кнопку **«Открыть папку»** (Open Folder).
3. В открывшейся папке создайте новую папку с именем **`chrome`** (если её там ещё нет).
4. Скопируйте файл `custom_extensions_panel.uc.js` из папки `chrome` этого архива в созданную вами папку `chrome` профиля Firefox.

---

## Шаг 2: Включение поддержки стилей и скриптов

1. В Firefox перейдите по адресу `about:config`.
2. Подтвердите предупреждение о рисках.
3. В строке поиска вверху введите:
   `toolkit.legacyUserProfileCustomizations.stylesheets`
4. Дважды кликните по найденному параметру, чтобы изменить его значение на **`true`**.

---

## Шаг 3: Установка JS-загрузчика (AutoConfig) в программу Firefox

Для запуска JS-скриптов в интерфейсе браузера необходимо скопировать файлы автоконфигурации в папку установки Firefox.

1. Найдите папку, в которую установлен Firefox на вашем компьютере. По умолчанию это:
   - `C:\Program Files\Mozilla Firefox\` (для 64-битной Windows)
   - `C:\Program Files (x86)\Mozilla Firefox\` (для 32-битной Windows)
2. Скопируйте файлы из папки `firefox-customization` в папку установки Firefox следующим образом:
   - Файл `config.js` поместите **в корень** папки Firefox (например: `C:\Program Files\Mozilla Firefox\config.js`).
   - Папку `defaults` скопируйте целиком в папку Firefox так, чтобы файл `autoconfig.js` оказался по пути:
     `C:\Program Files\Mozilla Firefox\defaults\pref\autoconfig.js`

> ⚠️ **Важно:** Для копирования файлов в системную папку `Program Files` могут потребоваться права администратора. Предыдущий файл `config-prefs.js` (если вы его создавали) можно удалить, так как теперь используется именно `autoconfig.js`!

---

## Шаг 4: Сброс кэша запуска и запуск

Поскольку Firefox кэширует конфигурационные файлы для быстрого старта, необходимо принудительно очистить кэш при первом запуске:

1. Вернитесь в окно Firefox на страницу `about:support`.
2. В правом верхнем углу найдите блок **«Службы поддержки»** (Try clearing the startup cache) и нажмите кнопку **«Очистить кэш запуска...»** (Clear startup cache...).
3. Подтвердите перезапуск браузера.

---

После перезагрузки:
- Стандартная кнопка расширений (серый пазл) будет скрыта.
- Появится новая кнопка расширений, которая открывает красивую современную панель управления.
- При клике средней кнопкой мыши (колёсиком) по новой кнопке откроется стандартная страница `about:addons`.
