## 1. Установка

### Временно (разработка / тест)

1. Firefox → `about:debugging#/runtime/this-firefox`
2. **Загрузить временное дополнение…**
3. Выбери `manifest.json` из папки расширения
4. Иконка появится на панели

> Временные дополнения пропадают после перезапуска Firefox. Загрузи снова или ставь с AMO / подписанный пакет.

### Из ZIP

В архиве в **корне** должен лежать `manifest.json` (не клади `firefox-customization/` и вложенные копии вроде `photopea/`).

## 2. Быстрый старт

1. Открой сайт с картинкой.
2. **Правый клик** по изображению.
3. Меню **🎨 Photopea**:
   - **Открыть изображение** — картинка как документ в Photopea.
   - **На холст: …** — холст размера пресета, размещение картинки (см. [режимы](#7-режимы-размещения)), по центру, вид вписан в экран.
4. Редактируй в Photopea.
5. Чтобы сохранить файл на диск — **Экспорт** (см. [§8](#8-экспорт-из-photopea)).

## 3. Popup на панели

Кликни иконку расширения.

| Элемент | Действие |
|---------|----------|
| **Открыть Photopea** | Новая вкладка с Photopea |
| **Список пресетов** | Клик → **пустой** холст этого размера |
| **Недавние изображения** | Повторно открыть недавно отправленный URL (если ещё доступен) |
| **Экспорт** (только на вкладке Photopea) | Скачать документ PNG / JPG / PSD |
| **Настройки** | Полная страница опций |

В popup указана клавиша: **Alt+Shift+P** (последнее изображение).

## 4. Контекстное меню

### На картинках / видео

ПКМ по image или video:

| Пункт | Смысл |
|-------|--------|
| **🎨 Photopea → Открыть изображение** | Загрузить в Photopea |
| **🎨 Photopea → На холст: Имя (Ш×В)** | Холст пресета (только **включённые** пресеты) |

### На странице Photopea

ПКМ на [photopea.com](https://www.photopea.com), когда документ открыт:

| Пункт | Смысл |
|-------|--------|
| **Экспорт из Photopea → PNG / JPG / PSD** | Экспорт через API Photopea → «Сохранить как» |

## 5. Настройки — Основные

Открытие: popup → **Настройки**, или `about:addons` → дополнение → **Параметры**.

### Язык интерфейса

| Вариант | Поведение |
|---------|-----------|
| **Авто (язык браузера)** | Как язык Firefox, если есть перевод |
| **Конкретный язык** | Жёстко задаёт язык меню, popup, настроек и уведомлений |

Смена языка применяется сразу.  
Имя расширения в `about:addons` по-прежнему от языка браузера (ограничение Firefox).

### Открывать в

| Вариант | Поведение |
|---------|-----------|
| **Новой вкладке** | Всегда новая вкладка Photopea |
| **Текущей вкладке** | Текущая вкладка заменяется на Photopea |

### Размещение изображения

Как картинка кладётся на **холст пресета** (для «Открыть изображение» не используется).  
См. [§7](#7-режимы-размещения).

### Фон холста

Цвет пустых областей:

- **Белый**
- **Прозрачный**
- **Чёрный**

### DPI по умолчанию

DPI для **новых** пресетов. Уже созданные пресеты хранят свой DPI.

### Уведомление, если не удалось извлечь изображение

Если включено — Firefox покажет уведомление. Можно вручную: **Копировать изображение** → Photopea → **Ctrl+V**.

## 6. Настройки — Пресеты холста

До **15** размеров.

Для каждого пресета:

| Поле | Смысл |
|------|--------|
| **Вкл / Выкл** | В меню и popup только **активные** |
| **Иконка** | Эмодзи / символ в меню |
| **Название** | Например «Instagram Story» |
| **Ширина / Высота** | Размер холста в px |
| **DPI** | Метаданные документа в Photopea |
| **✕** | Удалить (минимум один пресет) |

Кнопки:

- **+ Добавить пресет**  
- **Сохранить** — записать пресеты и общие настройки  
- **Сбросить по умолчанию** — 5 стандартных пресетов + дефолтные настройки  

По умолчанию: Full HD, Instagram Post, Instagram Story, A4 (300 DPI), YouTube Thumbnail.

## 7. Режимы размещения

Настройка: **Размещение изображения** (Основные).

Работает при **«На холст: …»**.

| Режим | Что делает | Когда удобно |
|-------|------------|--------------|
| **По центру (исходный размер)** | Холст = пресет. Картинка **без масштаба**, по центру. Большая — обрежется краями; маленькая — на фоне. | Нужен точный размер холста и пиксели как есть |
| **Вписать в холст** | Масштаб **пропорционально**, картинка **целиком** внутри (возможны поля). | Весь объект виден, без обрезки |
| **Заполнить холст (обрезка)** | Масштаб **пропорционально**, чтобы **закрыть** весь холст; лишнее срежется. | Сторис / обложки без полос |
| **Растянуть на холст** | Ровно в ширину и высоту (пропорции **могут исказиться**). | Редко; важны только точные px |

### Примеры

- Фото с телефона → **Instagram Story** → **Вписать** (всё видно) или **Заполнить** (без полос).  
- Иконка 64×64 → **Full HD** → **По центру**.  
- Жёстко 1280×720 любой ценой → **Растянуть**.

После размещения расширение:

1. Центрирует слой  
2. Вызывает **вписать вид** (`app.UI.fitTheArea`), чтобы не зумить вручную

## 8. Экспорт из Photopea

После правок:

1. Оставайся на вкладке Photopea.  
2. Либо:
   - **popup** расширения → **PNG / JPG / PSD**, либо  
   - ПКМ по странице → **Экспорт из Photopea** → формат.  
3. Укажи файл в диалоге сохранения.

Нужен открытый документ. Используются `saveToOE` и право `downloads`.

## 9. Горячая клавиша

| Клавиши | Действие |
|---------|----------|
| **Alt+Shift+P** (по умолчанию) | Открыть в Photopea **последнее** изображение, по которому был ПКМ через расширение |

Смена:

1. `about:addons`  
2. Шестерёнка → **Управление горячими клавишами расширений**  
3. **Send to Photopea** → свои клавиши  

Если картинки ещё не было — уведомление; Photopea может открыться пустым.

## 11. Разрешения

| Право | Зачем |
|-------|--------|
| `contextMenus` | Меню ПКМ |
| `storage` | Пресеты, настройки, недавние, язык |
| `tabs` / `activeTab` | Photopea, захват вкладки |
| `notifications` | Ошибки |
| `downloads` | Экспорт на диск |
| `<all_urls>` | Скачать картинку с любого сайта |
| `webRequest` + `webRequestBlocking` | Обход CORS на части сайтов (в т.ч. AMO) |

Аналитики нет. Сбор данных: **нет**.

## 12. Советы и ограничения

**Советы**

- Включай только нужные пресеты — меню короче.  
- Если сайт не отдаёт картинку: **Копировать изображение** → Photopea → **Ctrl+V**.  
- Для сторис — **Вписать** или **Заполнить**; для «холст N×M, пиксели как есть» — **По центру**.  

**Ограничения**

- На `addons.mozilla.org` Firefox **блокирует content scripts**; расширение пробует сеть / захват вкладки.  
- Очень большие файлы могут упереться в лимит длины URL.  
- Временные дополнения сбрасываются после перезапуска Firefox.  
- Папка `firefox-customization/` **не** часть WebExtension.

## 13. Если что-то не работает

| Проблема | Что сделать |
|----------|-------------|
| ПКМ ничего не даёт | Reload в `about:debugging`. Есть ли пункт **🎨 Photopea**? |
| Пустой холст / неверный размер | Актуальная сборка 1.1; проверь **Размещение** и что пресет **включён**. |
| Картинка в углу / огромный зум | Reload; «На холст» должен центрировать и вписать вид. Вручную — zoom fit в Photopea. |
| Экспорт молчит | Вкладка **Photopea**, документ открыт. Переустанови temporary add-on (право **downloads**). |
| Язык не меняется | Настройки → Язык → конкретный язык, не только Авто. |
| Сайт режет картинку | Копировать → вставить в Photopea. |

Консоль: `about:debugging` → Inspect → строки `STP:`.
